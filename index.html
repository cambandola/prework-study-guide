HTML

  • HTML defines the structure of the content on a webpage.
  • p is used for paragraphs in a body element.
  • h1-h6 are used as headers for your webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is used to add style and color to a webpage, so it doesn't look bland.
  • When using external CSS you have to add a link with an href tag that tells the program where to pull from.

Git

  • git status tells you what branch we are working on.
  • git checkout -b bname creates a new branch for us to work in.
  • git add -A is used to commit all files in working branch.

JavaScript

  • Arrays are zero-indexed and sequential.
